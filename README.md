# Personal WordPress Theme

Personal is a simple, clean, and elegant WordPress theme designed for personal blogs, portfolios, or anyone looking to share their thoughts online. This theme is fully responsive, lightweight, and highly customizable, making it an ideal choice for anyone looking to create a personal website or blog.

# Features
- Responsive Design: Looks great on mobile, tablet, and desktop devices.
- Customizable: Easily customize the header, footer, fonts, colors, and more through the WordPress Customizer.
- Theme Options: Simple and user-friendly settings for adjusting layout and style.
- Clean Code: Optimized for performance and easy to modify.
- SEO Friendly: Built with clean, semantic code for better search engine visibility.
- Support for Custom Widgets: Add widgets to your sidebar and footer easily.
- Support for Custom Menus: Create custom navigation menus.
- Social Media Integration: Easily add social media links to your site.

# Installation
Download the Theme: Download the personal-theme.zip file.
1. Install via WordPress Admin:
2. Go to your WordPress admin panel.
   - Navigate to Appearance > Themes.
   - Click Add New, then Upload Theme.
   - Choose the personal-theme.zip file and click Install Now.
   - Once installed, click Activate to set the theme live.
3. Customize: Go to Appearance > Customize to configure the theme options to your liking.

# Theme Customization
After activation, you can customize the theme settings:
- Site Identity: Change the site title and tagline.
- Colors: Customize the color scheme of your website.
- Fonts: Set the typography for the header, body, and other elements.
- Header: Upload your logo and choose a header layout.
- Footer: Customize footer text and widget areas.
# Widgets
This theme supports the following widgets:
- Recent Posts: Display a list of your latest blog posts.
- Categories: Show your site categories.
- Search: Add a search bar to your sidebar.
- Custom Text: Add custom HTML or text.
